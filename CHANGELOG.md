# 更新日志 (Changelog)

所有关于 **Ting Reader Client** 的重要变更都将记录在此文件中。

## [1.0.5] - 2026-03-04

### 🐛 修复与改进
- **章节播放逻辑优化**：修复了存在番外章节时，正文/番外切换不稳定的问题。现在播放列表严格跟随当前选中的标签页（正文或番外），不再自动跳转。
- **章节编号修正**：配合后端更新，解决了正文章节编号因番外插入而不连续的问题。

## [1.0.4] - 2026-03-03

### 🛠️ 后端适配与修复
- **深度适配新版后端数据结构**：修复了设置页面（播放偏好、外观设置等）无法正确保存或回显的问题，解决了因字段命名规范（驼峰/下划线）不一致导致的配置失效。
- **修复“客户端自动下载”配置**：修正了离线自动下载功能的读取逻辑，适配后端最新的嵌套配置结构 (`settings_json`)，现在该开关可以正确保存并生效。

### 💾 缓存管理优化
- **封面显示修复**：解决了缓存管理界面及离线播放模式下，书籍封面图片无法正常加载的问题。
- **缓存清理策略升级**：优化了文件清理逻辑，现在删除书籍缓存或批量删除所有章节时，会自动同步删除本地存储的封面图片，彻底释放磁盘空间。

### ⚡ 交互体验改进
- **剪贴板功能修复**：修复了设置页面中 Widget 嵌入代码“一键复制”按钮因焦点冲突导致报错（`NotAllowedError`）的问题。

## [1.0.3] - 2026-02-17

### ⚡️ 性能优化
- **离线体验**：优化离线登录后的缓存播放体验，修复了章节列表时长显示异常的问题。

## [1.0.2] - 2026-02-15

### ✨ 新增功能
- **离线能力增强**：实现了本地缓存管理，支持离线登录和播放已缓存的书籍。
- **界面优化**：优化了空状态下的界面显示，保持了桌面端宽敞的布局风格。

## [1.0.1] - 2026-02-08

### 🛠️ 修复与优化
- **前端路由修复**：修复了在非根路径下刷新页面导致白屏（`MIME type error`）的问题，将前端构建 `base` 路径调整为绝对路径 `/`。

## [1.0.0] - 2026-02-06

### 🎉 首次发布
Ting Reader 桌面客户端正式版，专为自托管有声书爱好者打造。

### ✨ 核心特性
- **全平台支持**：提供 Windows (.exe), macOS (.dmg), Linux (.AppImage/.deb/.rpm) 安装包。
- **无缝连接体验**：
    - **自托管友好**：自动处理 HTTPS 自签名证书与混合内容问题。
    - **智能重定向**：完美支持内网穿透/DDNS 环境下的 302/301 跳转登录，自动修正 POST 请求降级问题。
    - **零配置跨域**：内置 CORS 规避策略，无需服务端额外配置。
- **极致播放体验**：
    - **离线缓存**：自动缓存已播放章节，支持断网播放与秒级加载。
    - **精准 Seek**：重写本地文件服务逻辑，完美支持缓存文件的拖拽进度条（Range Request）。
    - **智能管理**：采用 LRU 策略自动管理缓存空间（默认 2GB），无需手动清理。

### 🛠️ 系统集成
- 自动记忆服务器地址与登录状态。
- 自动检测并修复历史遗留的无效地址配置。
- 深度适配 Windows/macOS 系统原生窗口样式。
