# 更新日志 (Changelog)

所有关于 **Ting Reader Client** 的重要变更都将记录在此文件中。

## [1.0.2] - 2026-02-15

### ✨ 新增功能
- **离线能力增强**：实现了本地缓存管理，支持离线登录和播放已缓存的书籍。
- **界面优化**：优化了空状态下的界面显示，保持了桌面端宽敞的布局风格。

## [1.0.1] - 2026-02-08

### 🛠️ 修复与优化
- **前端路由修复**：修复了在非根路径下刷新页面导致白屏（`MIME type error`）的问题，将前端构建 `base` 路径调整为绝对路径 `/`。

## [1.0.0] - 2026-02-06

### 🎉 首次发布
Ting Reader 桌面客户端正式版，专为自托管有声书爱好者打造。

### ✨ 核心特性
- **全平台支持**：提供 Windows (.exe), macOS (.dmg), Linux (.AppImage/.deb/.rpm) 安装包。
- **无缝连接体验**：
    - **自托管友好**：自动处理 HTTPS 自签名证书与混合内容问题。
    - **智能重定向**：完美支持内网穿透/DDNS 环境下的 302/301 跳转登录，自动修正 POST 请求降级问题。
    - **零配置跨域**：内置 CORS 规避策略，无需服务端额外配置。
- **极致播放体验**：
    - **离线缓存**：自动缓存已播放章节，支持断网播放与秒级加载。
    - **精准 Seek**：重写本地文件服务逻辑，完美支持缓存文件的拖拽进度条（Range Request）。
    - **智能管理**：采用 LRU 策略自动管理缓存空间（默认 2GB），无需手动清理。

### 🛠️ 系统集成
- 自动记忆服务器地址与登录状态。
- 自动检测并修复历史遗留的无效地址配置。
- 深度适配 Windows/macOS 系统原生窗口样式。
